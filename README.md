# Final Console

从设计到开发、测试、部署，打造企业级IT管理一站式解决方案。

## 微服务

### Final Sso

SSO（Single Sign On）即单点登录，是指在多个应用系统中，用户只需要登录一次就可以访问所有相互信任的应用系统。

### Final Admin

管理台，用于管理菜单
